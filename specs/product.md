# NewClinic

## Objetivo

Sistema web para agendamento de consultas entre pacientes e clínicas.

## Perfis de usuário

### Paciente

- Criar conta
- Buscar clínicas
- Solicitar agendamentos
- Receber notificações

### Clínica

- Criar conta
- Configurar disponibilidade
- Receber solicitações
- Aceitar ou recusar agendamentos

## Funcionalidades

- Cadastro
- Login
- Busca de clínicas
- Agendamento
- Notificações
- Gestão de disponibilidade
