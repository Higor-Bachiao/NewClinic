# Autenticação

## Cadastro de Paciente

Campos:

- Nome Completo
- Username
- CPF
- Email
- Senha
- Confirmar Senha
- Foto de Perfil

## Cadastro de Clínica

Campos:

- Nome da Clínica
- Nome do Dono
- CNPJ
- Email
- Senha
- Confirmar Senha
- Foto de Perfil
- Especialidade

## Login

Antes do login o usuário deve selecionar:

- Sou Paciente
- Sou Clínica

Após selecionar:

- Email
- Senha
